--- layout: doc title: deploy-vdoc description: | this task can deploy all VDoc jars into a repository permalink: /deploy-vdoc/ category: doc weight: 2 ---

Full name

com.vdoc.maven:vdoc-maven-plugin:{{vdoc_maven_plugin_version}}:deploy-vdoc

Attributes

Parameters

buildDirectory
Directory containing the generated JAR.
  • java.io.File
  • ${project.build.directory}
earFolder
the VDoc ear folder (prefer the configurator's ear to avoid upload not needed jar)
  • java.io.File
jarName
Name of the generated JAR.
  • java.lang.String
  • ${project.build.finalName}
mavenHome
the maven home folder
  • java.io.File
  • ${env.M2_HOME}
onlyParentPom
set to true for only deploy parents pom
  • boolean
  • false
project
The Maven project.
  • org.apache.maven.project.MavenProject
  • ${project}
repositoryId
the repository id used if password is needed for upload
  • java.lang.String
repositoryUrl
the repository url
  • java.lang.String
session
The Maven session.
  • org.apache.maven.execution.MavenSession
  • ${session}
targetGroupId
the VDoc group id
  • java.lang.String
  • com.vdoc.engineering
targetVersion
the VDoc upload version
  • java.lang.String
uniqueVersion
snapshot must used unique version
  • boolean
  • true
