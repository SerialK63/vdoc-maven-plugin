--- layout: doc title: Hard deploy description: this goal is used to deploy a project into the targeted VDoc install. permalink: /hard-deploy/ category: doc weight: 2 ---

Full name

com.vdoc.maven:vdoc-maven-plugin:{{vdoc_maven_plugin_version}}:hard-deploy

Attributes

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: runtime.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: package.

Required Parameters

vdoc.home
File the target VDoc instance home.

Optional Parameters

vdoc.home
File the target VDoc instance home.
with.custom
boolean the custom folder must be updated.default : true
include.test
boolean test jar should be deployed.default : true
include.source
boolean source jar should be deployed.default : false
sourceCustom
File Directory containing the generated source JAR.default : ${project.build.directory}
jarDirectory
File Directory containing the generated JAR.default : ${project.build.directory}
jarName
File Name of the generated JAR.default : ${project.build.finalName}
