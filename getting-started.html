VDoc maven repository

The plugin is actually not published on standard Nexus. You can find it into VDoc Nexus

Add VDoc repository




Configure username/password


VDoc repository is private you need to request username/password on MyVDoc


  You need help to configure username/password see Maven servers documentation



Plugin visioning


To simplify plugin usage the version number will follow VDoc version's (only major versions for example : 14.1).


Plugin minor version will be used for plugin update et bug fix.



  No Git tag will be created for plugin version only for VDoc version. If you find bug please check on last plugin version.
